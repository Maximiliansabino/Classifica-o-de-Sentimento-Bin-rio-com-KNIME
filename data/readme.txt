Sentiment Labelled Sentences Data Set
======================================

Fonte: UCI Machine Learning Repository
URL: https://archive.ics.uci.edu/ml/datasets/Sentiment+Labelled+Sentences

Descrição:
-----------
O dataset contém frases rotuladas com sentimento positivo (1) ou negativo (0),
extraídas de três fontes:
- Amazon (avaliações de produtos): 1000 frases
- Yelp (avaliações de serviços): 1000 frases
- IMDb (críticas de filmes): 1000 frases

Total: 3000 frases (1500 positivas + 1500 negativas)

Formato:
--------
Cada arquivo é separado por tabulação (TSV):
  sentença[TAB]rótulo

Onde rótulo é:
  0 = sentimento negativo
  1 = sentimento positivo

Referência:
-----------
Kotzias, D., Denil, M., De Freitas, N. and Smyth, P., 2015.
From Group to Individual Labels using Deep Features.
KDD 2015.

NOTA: Os arquivos neste repositório contêm apenas amostras representativas.
Para o dataset completo (3000 frases), baixe diretamente do UCI Repository.
